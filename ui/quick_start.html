Quick Start

  • 1. Install Background Watcher as Windows LogOn Task
    • Options > [x] Install Watchman as Windows logon task
    • This tiny background process will do all the work for you. The application you see here is just the GUI to configure that background watchman and can be safely closed at any time. You can stop the background watcher either in this GUI or thru your Windows/Computer management/Planned Tasks/SimmonWatcher_StartUp Windows-Task

  • 2. Use the top left [ + ] button to add a Profile.Use an example profile to get started.
  • 3. Define executables to watch. These will trigger your Simmon-Task's when the Executables are stopped or started.
  • 4. Define tasks to execute once the start or stop of one of the executables is detected.
